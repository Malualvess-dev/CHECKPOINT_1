ANÁLISE E DESENVOLVIMENTO DE SISTEMAS

O Curso

A crescente valorização dos profissionais Full Stack na área de TI enfatiza a importância do desenvolvimento de novas habilidades técnicas e conhecimentos de negócios, principalmente em um contexto ágil. Aqui a gente vai preparar você para isso.

Comece sua jornada adotando metodologias eficazes, como o Aprendizado Baseado em Projetos (Project Based Learning - PBL) e o Aprendizado Ágil (Agile Learning). Abordagens práticas para soluções de problemas tangíveis, voltadas para produtos e serviços de grandes empresas.

Você vai construir projetos com conceitos avançados de desenvolvimento de software, desde a programação de backend e frontend até testes, modelagem e programação de Bancos de Dados, tanto relacionais quanto não relacionais. Criar aplicativos móveis para plataformas Android e iOS. E explorar tecnologias como Inteligência Artificial, Internet das Coisas e DevOps.

A colaboração com empresas líderes em tecnologia, como Oracle, Microsoft, IBM e AWS, é mais um diferencial desta Graduação. Assim, você vai colocar em prática habilidades diretamente relacionadas a essas plataformas, impulsionando sua carreira em um mercado supercompetitivo.

E como um Full Stack do futuro também precisa dominar skills de empreendedorismo, você vai criar sua própria startup por meio do programa Startup One.

Você vai se preparar para:

Modelar sistemas utilizando Análise Orientada a Objetos e melhores práticas de desenvolvimento (UML, XP, Scrum). Construir projetos de aplicações sob medida (distribuída, web e mobile) aplicando Design Thinking & Prototyping para um cliente real e atualizando sistemas já existentes. Integrar e desenvolver aplicações web com interfaces responsivas utilizando padrões W3C, web semântica e SEO. Usar tecnologias como HTML 5, CSS 3, Javascript, React JS e NextJS. Desenvolver aplicações utilizando as principais ferramentas e linguagens de programação, interagindo com bancos de dados padrão SQL (Oracle), Big Data & Analytics e com as interfaces do sistema (camadas MVC). Criar aplicações com os principais frameworks de mercado das plataformas Java e .Net (JPA, Hibernate, EJB, ASP.NET MVC, WEB API e Entity Framework). Apoiar as estratégias de negócios digitais desenvolvendo aplicações com JSF, Spring, web services e cloud computing (IBM Cloud), tecnologia WebServices e RESTful (JAX-WS, JAX-RS, XML, JSON). Construir dashboards e criar aplicações para Internet das Coisas, Inteligência Artificial e Smart Cities.
